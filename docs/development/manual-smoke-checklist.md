# Manual Smoke Checklist (Pre-merge)

Use this checklist before merging frontend changes in this branch.

- [ ] Add recipe to meal plan
- [ ] Remove recipe from meal plan
- [ ] Generate groceries from meal plan
- [ ] Open grocery list without meal plan underneath
- [ ] Share recipe text
- [ ] Copy recipe text
- [ ] Print / Save as PDF from desktop
- [ ] Print / Save as PDF from phone

## Regression: Manual "Write from scratch" persistence

- [ ] Add → Write from scratch
- [ ] Title: `Test`
- [ ] Fill servings, prep time, and cook time
- [ ] Add at least one ingredient
- [ ] Add at least one instruction
- [ ] Save recipe
- [ ] Refresh page
- [ ] Open saved recipe
- [ ] Verify title and details still appear after refresh
